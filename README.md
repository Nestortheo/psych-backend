Django backend for a psychologist website.  
Handles form submissions from the React frontend and sends emails through the backend.

## Tech Stack
- Python
- Django
- Django REST Framework
- Email handling (SMTP)
- SQLite/PostgreSQL

## Features
- Backend API for frontend communication
- Form submission handling
- Email sending functionality
- Django admin panel
- Structured backend architecture

## How to run locally

pip install -r requirements.txt  
python manage.py migrate  
python manage.py runserver
